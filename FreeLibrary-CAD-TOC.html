CAD LISP
도면 리스트 자동화

← Back to Library

시트폼 기준을 먼저 설정한 다음 같은 폼을 스캔해 도면 리스트를 반자동으로 생성하는 AutoLISP 도구입니다. 핵심 명령어는 TOCFORMSET, TOCFORMSCAN입니다.

Command Summary

Step 01 Command
Setup

TOCFORMSET

도면 리스트를 만들 기준 시트폼을 선택해 폼 기준을 저장합니다. 같은 프로젝트에서 한 번 먼저 실행한 뒤 스캔 명령으로 넘어갑니다.

  • 명령창에 TOCFORMSET 입력
  • 기준이 될 시트폼을 선택
  • 폼 기준 저장 후 스캔 준비
Step 02 Command
Scan

TOCFORMSCAN

저장된 시트폼 기준과 같은 폼들을 찾아 도면 리스트를 생성합니다. 도면명이 정리된 상태에서 실행하면 리스트 작성 시간을 줄일 수 있습니다.

  • 명령창에 TOCFORMSCAN 입력
  • 같은 폼 영역을 스캔
  • 스캔 결과로 도면 리스트 생성
