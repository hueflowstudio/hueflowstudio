도면 리스트 자동화

도면 시트폼 안에 있는 도면번호, 도면명 같은 작은 칸들을 기준으로 잡고, 같은 폼을 가진 시트들을 스캔해 도면 리스트를 만드는 AutoLISP 도구입니다. 먼저 TOCFORMSET으로 폼과 칸 위치를 지정하고, 이후 TOCFORMSCAN으로 리스트를 생성합니다.

사용 순서

01 Load

리습 로드

AutoCAD에서 APPLOAD를 실행한 뒤 cad_toc_auto.lsp 파일을 선택해 로드합니다.

02 Set

폼 기준 설정

명령창에 TOCFORMSET을 입력한 뒤, 도면 리스트를 만들 기준 시트폼 전체를 선택합니다. 이 단계에서 프로그램이 "이 모양의 시트폼을 기준으로 찾겠다"는 기준을 저장합니다.

  • 명령어: TOCFORMSET
  • 먼저 기준이 될 시트폼 외곽 또는 폼 객체를 선택
  • 같은 프로젝트에서 기준 폼을 바꿀 때만 다시 설정
03 Pick

폼 안의 리스트 칸 선택

기준 폼 안에서 프로그램이 읽어야 할 작은 박스들을 차례대로 선택합니다. 어제 테스트한 흐름처럼 도면 리스트에 들어갈 칸을 박스로 찍어주는 단계입니다.

  • 도면번호 칸 선택
  • 도면명 또는 시트명 칸 선택
  • 축척, 날짜, 비고 등 필요한 칸 선택
  • 리스트가 들어갈 위치 또는 기준 칸 선택
04 Text

텍스트 옵션 입력

프롬프트가 나오면 리스트에 사용할 글자 크기 같은 값을 입력합니다. 도면 스케일에 맞춰 기존 표기와 비슷한 크기로 맞추면 됩니다.

  • 텍스트 높이 입력
  • 필요 시 행 간격이나 시작 위치 확인
  • 입력값은 도면 표기 크기에 맞춰 조정
05 Scan

도면 리스트 생성

TOCFORMSCAN을 실행하면 저장된 폼 기준과 칸 위치를 바탕으로 같은 시트폼들을 찾아 도면 리스트를 생성합니다.

  • 명령어: TOCFORMSCAN
  • 같은 폼을 가진 도면 시트들을 스캔
  • 선택해 둔 칸에서 도면번호와 도면명 정보를 읽음
  • 지정된 위치에 도면 리스트 생성

명령어 요약

Step 01 Command
Setup

TOCFORMSET

기준 시트폼과 폼 안의 정보 칸 위치를 저장하는 명령어입니다. 도면번호, 도면명 등 읽어야 할 칸을 차례대로 선택하는 단계가 포함됩니다.

  • 명령창에 TOCFORMSET 입력
  • 기준 시트폼 선택
  • 폼 안의 작은 정보 칸들을 순서대로 선택
  • 텍스트 높이 등 옵션 입력
  • 설정 완료 후 스캔 준비
Step 02 Command
Scan

TOCFORMSCAN

TOCFORMSET에서 저장한 폼 기준으로 같은 시트들을 찾아 도면 리스트를 생성하는 명령어입니다. 폼 기준을 먼저 설정한 뒤 실행합니다.

  • 명령창에 TOCFORMSCAN 입력
  • 같은 시트폼을 가진 도면들을 스캔
  • 선택한 칸에서 도면번호, 도면명 등 정보 읽기
  • 스캔 결과를 도면 리스트로 작성
