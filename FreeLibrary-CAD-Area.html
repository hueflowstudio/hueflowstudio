면적/평수 표기

← Back to Library

닫힌 폴리라인을 선택하면 면적과 평수 텍스트, 합계표를 자동 생성하는 AutoLISP 도구입니다. 현재 버전은 명령어 HFA로 실행합니다.

Version History

v2.0 2026.05.14
Recommended

객체 스냅 복원 안정화 업데이트

면적/평수 표기 작업 후 기존 도면 작업 환경이 안정적으로 유지되도록 복원 로직을 보강한 버전입니다.

  • 작업 종료 후 객체 스냅(OSNAP)이 비활성화되는 현상을 수정했습니다.
  • 명령 취소 또는 오류 발생 시에도 OSNAP, CMDECHO 설정이 기존 값으로 복원되도록 안전 처리 로직을 추가했습니다.
  • 도면 갱신을 위한 REGEN 자동 실행을 더해 결과 표시 안정성을 개선했습니다.

v1.0 2026.05.13

LINE + MTEXT 기반 표기 버전

닫힌 폴리라인의 면적과 평수를 표기하고, 전체 합계표를 도면에 직접 생성하는 초기 배포본입니다.

  • 명령어 HFA로 실행
  • TABLE 객체 대신 LINE + MTEXT 조합으로 표 생성
  • 면적, 평수, 합계 표기 흐름 포함
