면적/평수 표기

← Back to Library

닫힌 폴리라인을 한 번 선택하면 각 영역에 면적/평수 라벨을 쓰고, 합계표까지 자동 생성하는 AutoLISP 도구입니다.

HF_AREA.lsp 다운로드

사용법

v6.0 Latest
Recommended

한 번 선택으로 면적 라벨 + 합계표 생성

HFA
  • 닫힌 폴리라인을 창 선택 또는 개별 선택 후 Enter를 누릅니다.
  • 각 폴리라인 오른쪽 아래에 면적/평수 라벨이 자동으로 생성됩니다.
  • 합계표는 선택한 폴리라인 전체의 오른쪽 상단에 자동 배치됩니다.
Manual Option

합계표 위치만 직접 찍고 싶을 때

HFAT

면적 라벨은 동일하게 자동 생성하고, 합계표 왼쪽 상단 위치만 사용자가 직접 클릭합니다.

자동 로드

Auto Tip

APPLOAD를 매번 하지 않으려면

  • AutoCAD의 acaddoc.lsp 또는 Startup Suite에 HF_AREA.lsp를 등록합니다.
  • 파일 위치를 옮기면 자동로드 경로도 다시 맞춰야 합니다.
  • 현재 Hueflow 기본 설치 경로는 Documents\HueflowCADTools\cad-area입니다.

주의사항

Check 01

닫힌 폴리라인만 계산

  • 열린 폴리라인은 면적 계산에서 제외됩니다.
  • 단위는 도면이 mm 기준이라는 전제로 m2와 PY를 계산합니다.
  • 기존 결과를 다시 만들 때는 이전 HF-AREA 레이어 결과물을 삭제한 뒤 실행하면 깔끔합니다.
