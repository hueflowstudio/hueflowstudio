입면 기준선 생성 기본 버전

가구부 입력 후 기준선, 치수 레이어, 꺾임 마커를 생성하는 AutoLISP 배포본입니다.

  • A-ELEV, A-ELEV-DIM, A-ELEV-MRK 레이어 자동 구성

Auto Tip

APPLOAD를 매번 하지 않으려면

  1. HUEFLOW_ELEV.lsp 파일을 계속 둘 폴더에 저장합니다.
  2. AutoCAD에서 APPLOAD 입력 후 Startup Suite 또는 시작하기 모음을 엽니다.
  3. Add 또는 추가 버튼으로 HUEFLOW_ELEV.lsp 파일을 등록합니다.
  4. AutoCAD를 다시 열거나 새 도면을 열어 HF2 명령어가 바로 먹는지 확인합니다.

이 작업은 한 번만 하면 됩니다. 파일 위치를 옮기면 Startup Suite에서 다시 등록해야 합니다.
